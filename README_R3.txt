高峰競賽平台｜唯一正式整合版 R3
版本：2026.07.16-R3

本版新增：
1. 首頁所有區塊均可顯示／隱藏與自由排序。
2. 支援上移、下移、移到最上、移到最下。
3. 首頁統計卡（今日加權、今日實收、今日件數、每日之星）可顯示／隱藏、改名稱、改圖示與排序。
4. 快捷功能整區可移到首頁任意位置；快捷鍵內部仍可自行排序。
5. 歷史資料新增「各隊業績報表」：可選日期、多選隊伍、選欄位、選排序。
6. 可一次匯出多隊 Excel。
7. PDF 採瀏覽器原生繁體中文字型列印版，開啟列印視窗後選「儲存為 PDF」，避免繁體中文亂碼。
