# 高峰競賽平台 V2.1

本版以 V2.0.2 為基礎整合：

- 競賽固定三個核心標準：加權保費、實收保費、A&H。
- 三項可個別啟用／停用。
- 支援全部達成（AND）或任一達成（OR）。
- 各職級可設定不同目標。
- 可設定競賽開始日與結束日。
- 可新增其他條件，例如件數、指定商品或商品分類。
- 儲存競賽後首頁立即重新計算，不需重新整理。
- 首頁會分別顯示完成值、目標、差額及達成率。
- 保留 V2.0.1 的通訊處進度、首頁方塊管理、匯率小數修正。
- 保留 V2.0.2 的獎勵活動日期與業績篩選規則。
- 外觀設定可修改加入主畫面的名稱、favicon 與 App 圖示。

## 更新 GitHub

將本資料夾內所有檔案上傳並覆蓋 GitHub Pages 專案根目錄，等待部署完成後強制重新整理。

iPhone 更換圖示後，需要先刪除原本主畫面捷徑，再從 Safari 重新加入主畫面。

## 資料儲存

本版本仍以 LocalStorage 為主要儲存方式。Firestore 正式分集合同步與 Excel 正式批次匯入仍屬後續階段。
