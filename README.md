# 高峰競賽平台 V2.1.2 管理架構整合版

## 本版完成
- 首頁設定只保留方塊顯示／隱藏與自訂方塊。
- 通訊處進度移入「競賽與目標管理」。
- 新高峰、新極峰及自訂競賽統一在同一處維護。
- 個人競賽可依職級設定加權保費、實收保費、A&H。
- 通訊處目標可設定日期、目標值，並選擇自動抓取或手動完成業績。
- 移除首頁設定內重複的通訊處及個人競賽手動欄位。
- 儲存競賽後首頁立即重新計算。

## 注意
資料目前仍以瀏覽器 LocalStorage 為主；Firestore 分集合同步尚未完成。


## V2.1.2 更新
- 新極峰固定啟用「實收保費」核心標準。
- 新高峰、新極峰共用加權保費、實收保費、A&H 架構。
- 舊資料升級時會自動補上新極峰實收欄位。
- 若某職級尚未填實收目標，首頁會顯示「尚未設定」，避免誤顯示為已達成。
