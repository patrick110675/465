智慧 Excel 業績匯入更新（2026-07-20）

1. 上傳含多工作表的 Excel 時，業績匯入會優先選擇「原始資料」，否則依欄位內容自動判定最適合的工作表。
2. 自動辨識：姓名、商品名稱／商品代碼、e投保上傳日期／報件日期、保費、實收、原始加權、加權計績、A&H。
3. Excel 已提供實收與各類計績時，直接使用 Excel 數值；未提供時才依平台商品設定自動計算。
4. 匯入前顯示可匯入、錯誤、重複筆數；錯誤列不影響其他正確資料。
5. 同一份檔案重複上傳會自動跳過；有時間戳記的同一筆資料可更新，不會重複累加。
6. 匯入後沿用原有報件資料流程，自動更新排行榜、主管業績、競賽進度與獎勵活動。
