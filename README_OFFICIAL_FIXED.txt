本版只整理並修正業績統計核心，沒有更動主題與版面。

統計規則：
1. 業代本人：實收、原始加權、競賽加權、A&H、件數皆計入。
2. 同組直屬主任：同步計入業代的實收、原始加權、競賽加權、A&H。
3. 件數只算實際報件本人，不加給主管。
4. 我的競賽進度與個人排行榜共用同一個統計核心與競賽期間。
5. 修改、刪除、回收桶還原後皆由現有報件即時計算，不保存另一份累積值，因此不會留下殘值。
6. 區單位、隊伍、組別、職級與商品排行榜仍以原始報件加總，避免主管同步造成團隊重複計算。

直屬主管判定：
- 報件人職級為「業代／新進業代／特定新進業代」；
- 與其同組、職級為「主任／新進主任」的人員視為直屬主管。
